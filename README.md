# How to Run Locally

- Download the ``` Nodejs ``` From [NodeJs Downloads](https://nodejs.org/en/download/)
- Install the ``` http-server ``` From the following command
 ```
 npm install -g http-server 
 ```
- Go to your project directory
 ```
 cd D:\path\to\your\project
  ```
- Start The Server 
 ```
 http-server
 ```
